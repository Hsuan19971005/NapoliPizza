# Napoli 披薩線上訂購

Napoli Pizza 是向拿波里披薩致敬的披薩訂購網站，具有以下幾個主要功能：
1. 分店查詢：縣市分店查詢 | 分頁功能
2. 菜單瀏覽：食物分類查詢
3. 訂單查詢：訂單查詢 | Captcha驗證碼
4. 線上訂購系統：餐點訂購 | 購物車

## Technologies 使用技術
- Front-end 前端 : HTML | CSS | JavaScript | Tailwind CSS | Cookie
- Back-end 後端 : PHP | Laravel 8
- Database 資料庫 : MySQL
Deployment 部署 : GCP
- Version Control 版本控制: Git | GitHub

## Version 版本
- PHP 8.3.2
- Laravel 8
- Node 16
